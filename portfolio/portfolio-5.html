OTP Based authorisation applicatin

A secure and efficient OTP-based authentication app built using React.js for seamless user login. This app generates a One-Time Password (OTP) that expires in 30 seconds, ensuring enhanced security. Users can request a new OTP by clicking on "Resend OTP" after expiration. Designed specifically for **1536x713 resolution**, this app provides an optimized experience on desktop and laptop screens.

Features

  • Secure OTP-based authentication
  • Automatic OTP expiration after 30 seconds
  • OTP regeneration on clicking "Resend OTP"
  • Optimized exclusively for **1536x713 resolution (desktop/laptop screens)**
  • Interactive and real-time OTP validation feedback
  • Modern and sleek UI using CSS

Highlights

  • Enhanced security with time-sensitive OTP
  • Dynamic UI powered by React.js
  • Designed specifically for **desktop/laptop screens**
  • Fast and efficient state management with React
