KodBook - #1 social media application

KodBook is a social media application designed to connect users through posts, comments, and interactions. It offers a seamless and engaging user experience with secure authentication, user profiles, and a follow system.

Features

  • Signing Up and Logging In: Secure user authentication with account creation and login functionalities.
  • Post Creation: Users can create and share posts with text and images.
  • Commenting: Engage with posts through comments.
  • User Profiles: View and edit personal profiles.
  • Follow System: Follow other users to see their posts in your feed.

Highlights

  • Secure User Authentication
  • Interactive and User-Friendly Interface
  • Responsive Design for All Devices
  • Optimized Performance and Scalability
  • Technologies Used: HTML, CSS, Java Spring
